## 🛠️ Tecnologias Utilizadas

Neste projeto foram utilizadas as tecnologias base do desenvolvimento web:
* **HTML5:** Para toda a estruturação semântica da página.
* **CSS3:** Para a estilização, criação do *Dark Theme*, layout e posicionamento dos elementos.
* **JavaScript:** Para interações dinâmicas da página (como ações de botões ou menu).

## 🌟 Sobre o Projeto

Este é um template simples e bem-humorado, ideal para treinar conceitos de estruturação web e estilização de interfaces. O design conta com:
* **Menu de Navegação:** Links para Home, Fale Conosco e Nossos Serviços.
* **Seção Hero:** Um texto de apresentação criativo para atrair clientes ("Resolvendo Seus Problemas Hidráulicos Com Estilo!").
* **Call to Action:** Um botão vermelho chamativo para contato.
* **Botão Flutuante:** Um ícone do WhatsApp fixo na tela para facilitar a comunicação.
