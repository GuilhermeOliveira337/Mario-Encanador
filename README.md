<h1 align="center">🍄 Mario Encanador</h1>

<p align="center">
  Projeto inspirado no universo do Mario, desenvolvido com HTML, CSS e JavaScript.
</p>

<hr>

<h2>📌 Sobre o projeto</h2>
<p>
Este projeto simula uma experiência interativa baseada no clássico Mario, trazendo elementos visuais e comportamentos que remetem ao jogo original.
</p>

<p>
O objetivo principal foi praticar desenvolvimento front-end de forma criativa, aplicando conceitos modernos de estruturação, estilização e interatividade.
</p>

<hr>

<h2>🚀 Tecnologias utilizadas</h2>
<ul>
  <li><b>HTML5</b> - Estrutura da aplicação</li>
  <li><b>CSS3</b> - Estilização e layout</li>
  <li><b>JavaScript</b> - Lógica e interatividade</li>
</ul>

<hr>

<h2>🎮 Funcionalidades</h2>
<ul>
  <li>Animações inspiradas no universo do Mario</li>
  <li>Interação com o usuário</li>
  <li>Layout responsivo</li>
  <li>Manipulação do DOM</li>
</ul>
